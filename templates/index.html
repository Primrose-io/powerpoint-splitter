How It Works
  1. Add tags to your PowerPoint slides by adding [tag1,tag2] at the end of slide notes
  2. Upload your presentation file
  3. The app creates separate presentations for each tag
  4. Each tagged slide appears only in presentations with matching tags
  5. Slides tagged with [*] appear in all presentations
